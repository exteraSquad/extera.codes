-We're getting ready to go on Google Play, and we needed to create a new key that allowed us to put our app in the store and there was no warning about a malicious app.
Also, we would have done it later anyway, since our last key was only for a year, and this one we did for 25 years

6.
-How does the Zalgo filter work?
-Zalgo filter checks every character in the message and username, if it sees that there is something in the given character that prevents it from being read, it removes it. This feature can affect the performance of the application.
